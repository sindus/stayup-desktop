What you get

All your updates, one place

Add any source in seconds and let StayUp do the rest.

🐙

GitHub Releases

Subscribe to any repository. Get release notes the moment a new version drops — from vercel/next.js to your favourite indie library.

▶️

YouTube Channels

Follow creators by channel handle. New videos appear directly in your feed without opening YouTube or dealing with its algorithm.

📡

RSS / Atom Feeds

Paste any feed URL — blogs, podcasts, news sites. If it speaks RSS, StayUp reads it.

🌐

Web Scraping

No RSS? No problem. Target elements with CSS selectors and track any page that doesn't publish a feed.

📚

Documentation Tracking

Monitor docs sites for changes, browse version history, and diff any two snapshots side by side.

⚡

Unified Feed

Everything sorted chronologically. No tabs, no switching apps — just one clean timeline with all your sources.

🔄 Auto-update

StayUp updates itself silently in the background. A banner notifies you when a new version is ready.

🌗 Light & dark theme

Follows your OS preference automatically. Switch manually any time from the user menu.

🌍 English & French

Full interface localisation included. More languages can be added via the translations module.

🔐 Account-based

Your feeds sync to your account — sign in with email/password or OAuth and pick up where you left off.

Get started

Install StayUp

Choose the package that fits your system.

🍎 macOS

Homebrew Cask
brew install --cask sindus/tap/stayup
Direct download
↓ Download .dmg (Apple Silicon / Intel)

🪟 Windows

Installer (recommended)
↓ Download .exe installer
MSI (enterprise)
↓ Download .msi

🐧 Linux — Debian / Fedora

Debian / Ubuntu (.deb)
sudo dpkg -i StayUp_*_amd64.deb
Fedora / RHEL (.rpm)
sudo rpm -i StayUp-*-1.x86_64.rpm

🐧 Linux — Universal

AppImage (any distro)
chmod +x StayUp_*_amd64.AppImage
./StayUp_*_amd64.AppImage
Snap
sudo snap install stayup

Uninstalling

macOS: drag StayUp from /Applications to Trash · brew uninstall --cask stayup
Windows: Settings → Apps → Installed apps → StayUp → Uninstall
Linux deb: sudo apt remove stayup · rpm: sudo rpm -e StayUp · snap: sudo snap remove stayup

How it's built

CI / CD

Every push is checked. Every release is verified across all platforms before publishing.

🔍 Continuous Integration

Runs on every push to main and on all pull requests. The build is blocked until everything passes.

TypeScript check ESLint Prettier Vitest

🚀 Release pipeline

Triggered by a v* tag. Builds native packages for all platforms in parallel, then publishes only when every job succeeds.
